# ADAPT Template Library
**Densight Labs — Applied AI. Not just talked about.**

Ready-to-use templates for every phase of the ADAPT Framework. Built for practitioners who need to ship AI products, not talk about them. Each template is battle-tested on real projects across Pakistan, GCC, and US markets.

## What Is Inside
- **Phase Templates:** Structured docs for Assess, Design, Architect, Prototype, Transform
- **Checklists:** Quality gates and deliverable trackers per phase
- **Stakeholder Briefs:** Pre-written update formats for technical and business audiences
- **Risk Registers:** Common failure modes and mitigation strategies

## ADAPT Phase: All Phases
These templates span the entire ADAPT Framework lifecycle. Use them as starting points to accelerate delivery and maintain consistency across AI initiatives.

## Template
**Use this when:** Starting any new AI project or standardizing existing workflows.
**Key Fields:**
- Project name and phase identifier
- Stakeholder list with roles
- Success criteria (3-5 measurable)
- Risk assessment matrix
- Timeline with milestones
- Resource allocation

**Example:**
Project: Retail demand forecasting | Phase: Architect | Stakeholders: CTO, Data Lead, Ops Manager | Success: <5% MAPE, 2-week deployment | Risks: Data quality (HIGH), API latency (MEDIUM)
